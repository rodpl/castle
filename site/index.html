Welcome
Castle is a small collection of tools and an Inversion of Control container intended to work together to make the develpment of applications easier.

While we don't pursue a paradigm shift, we do aim to make daily life tasks simpler and well organized. To reach this goal we rely on

  • Component oriented programming
  • Aspect oriented programming
  • Separation of concerns

Our subprojects help us delivering this capabilities to the CLI. They are totally independent and can be used alone as well.

Castle Container is a very simple inversion of control container which can combine facilities to grow orthogonally. For example, you can create a NHibernate facility thus your components can be automagically persisted, or use our AOP facility which will allow your components to use mixins and interceptors and so on. This facilities work side-by-side, hence the use of the word orthogonal, without interfering or depending on each other, to provide you a set of features.
